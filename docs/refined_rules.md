# Q群群规

为了维护良好的群聊环境，创造和谐友好的交流氛围，请所有成员自觉遵守以下规定：

## 一、禁止事项

1. 禁止讨论和传播任何与🪜、VPN等相关的内容
2. 禁止刷屏行为（如发送大量相同或相似的消息、表情包等导致消息超出屏幕范围）
3. 禁止讨论或传播以下违法违规内容：
   - 色情、赌博、毒品相关内容
   - 政治敏感话题
   - 血腥暴力内容
   - 谩骂、引战言论
   - 故意挑衅他人行为
   - 侵犯他人隐私（如开盒、开户等）行为
4. 严禁宣传网络游戏外挂、修改器等影响公平游戏和他人体验的不道德内容
5. 禁止滥用机器人AI对话功能，刷账单行为将受到严惩（正常调侃不在此列）
6. 禁止通过机器人传播黄赌毒、政治敏感话题、血腥暴力等违法违规内容
7. 未经群主同意，禁止宣传其他群聊

## 二、管理规定

1. 卖号行为需合理定价，禁止价格过于离谱
2. 如不能经常活跃，请在名字后添加标签（例如：开学别踢）
3. 禁止私自拉他人进群！违规者将直接被踢出群聊
   - 如需宣传，应在群内公开进行，而非私下拉人

## 三、处罚规则

1. 首次违反群规：管理员将撤回违规内容并警告，视情节轻重进行禁言
2. 再次违反群规：处以7天禁言处罚
3. 第三次违反群规：直接移出群聊
4. 屡教不改者：移出群聊并加入黑名单

## 四、特别说明

1. 关于滥用机器人的详细处罚规则，请访问：https://qbot.acmcdev.top/（在建）
2. 如有人邀请您进入其他群聊，请谨慎判断，本群对因加入其他群聊产生的任何后果概不负责

请所有成员共同维护群聊环境，谢谢配合！
